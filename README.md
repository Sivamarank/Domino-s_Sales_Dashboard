# Domino's Sales Dashboard – Power BI Project

## Project Overview

This project is a Power BI dashboard created to analyze Domino's sales performance, customer ordering behavior, and product category insights. The dashboard helps track important KPIs, identify sales trends, and support business decision-making using data-driven insights.

The report provides interactive visualizations for:

* Domino's sales performance analysis
* Pizza category-wise revenue insights
* Customer purchasing trends
* Best-selling pizza and product performance tracking
* Business KPI monitoring

---

# Tools & Technologies Used

* **Power BI** – Dashboard creation and visualization
* **Excel / CSV Dataset** – Data source
* **Power Query** – Data cleaning and transformation
* **DAX** – KPI calculations and measures

---

# Key Features

* Interactive dashboard with filters and slicers
* Category-wise sales analysis
* Revenue and order tracking
* Customer insights dashboard
* Dynamic KPI cards
* Trend analysis using charts and graphs

---

# Business Insights

The dashboard helps answer important business questions such as:

* Which pizza categories generate the highest revenue?
* Which pizzas and menu items perform best?
* What are the customer ordering patterns?
* Which categories need improvement?
* How do Domino's sales trends change over time?

---

# Dashboard KPIs

Some important KPIs used in this project:

* Total Sales
* Total Revenue
* Average Purchase Value
* Category-wise Pizza Sales
* Customer Count
* Top Performing Pizzas & Menu Items
